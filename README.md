# WastedLands contract

### development
```
  npm install

  truffle console
```

### testing
```
  truffle test
```

### verify

```
  truffle run verify WastedWarrior@{contract-address} --network testnet
```

### migrate
```
  truffle migrate
```

### deploy 
```
  truffle migrate --network testnet
```
### migrate specific file 
```
  truffle migrate -f 3 --to 3 --network testnet
```
